# 📖 Story Telling Website

An interactive, AI-powered Horror Story Generator that uses Google's Gemini API to create spine-chilling tales.

## 🚀 Features
- **AI Story Generation**: Generates unique horror stories based on user prompts.
- **Atmospheric Design**: Dark, immersive UI with horror-themed styling and smooth animations.
- **Interactive Chat**: A conversation-like experience to build a narrative.
- **Responsive Layout**: Designed for both desktop and mobile users.

## 🛠️ Built With
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend API**: Python (Flask) for processing requests.
- **AI Model**: Google Gemini API for natural language generation.

## 📦 Getting Started

### Prerequisites
- Python 3.9+ installed on your machine.
- A Google Cloud / Gemini API key.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/AHSAN-888/Story-telling-website.git
   cd Story-telling-website
   ```
2. Install dependencies (if any, typically Flask or Google Generative AI):
   ```bash
   pip install flask google-generativeai
   ```
3. Run the application:
   ```bash
   python main.py
   ```
4. Open your browser and navigate to `http://127.0.0.1:5000`.

## 🎨 Design Aesthetics
The website features deep blacks, vibrant horror-themed accents (neon greens/reds), and smooth transitions to provide a premium and immersive storytelling experience.
